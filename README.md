# Recently Added for Obsidian

Displays a list of your most recently added files in the left sidebar, sorted by creation time.

<img width="348" height="415" alt="image" src="https://github.com/user-attachments/assets/c7335c19-6563-4b7a-ab79-62ed1aead080" />

## Features

- Sidebar panel showing recently added files with age indicators (minutes, hours, days)
- Click to open, Ctrl-click to open in a new pane, right-click for the full Obsidian file menu
- Middle-click to open in a new tab
- Drag files directly from the panel to other locations
- Exclude files by path regex patterns
- Configure maximum number of files shown and age cutoff

## Installation via BRAT

1. Install the [BRAT plugin](https://github.com/TfTHacker/obsidian42-brat) from Obsidian Community Plugins
2. Open BRAT settings → **Add Beta Plugin**
3. Enter: `https://github.com/motion2082/recently-added-obsidian`
4. Enable the plugin in Settings → Community Plugins

## Usage

Open the panel from the Command Palette (`Ctrl+P`) → **Recently Added: Open**

<img width="714" height="146" alt="image" src="https://github.com/user-attachments/assets/e92474c9-5b72-4e72-a6d3-020f90e0af99" />

## Settings

| Setting | Default | Description |
|---|---|---|
| Max files shown | 50 | Maximum number of files to display |
| Max age (days) | 30 | Only show files created within this many days (0 = no limit) |
| Omitted path patterns | — | Regex patterns — matching file paths are excluded (one per line) |
